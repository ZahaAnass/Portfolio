## Installation Instructions

1. Clone the repository:

    ```bash
    git clone https://github.com/ZahaAnass/Portfolio.git
    ```

2. Navigate to the project directory:

    ```bash
    cd Portfolio
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

## Usage
